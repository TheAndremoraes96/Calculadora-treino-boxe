# 🥊 BoxTimer Pro Community v1.0

Sistema web para treinadores e atletas de boxe com cadastro, avaliação física, dashboard, histórico, Firebase, timer de rounds, feed, eventos e backup.

## Status
✅ Versão estável publicada no GitHub Pages.

## Funcionalidades
- Cadastro de atletas
- Firebase Firestore
- Dashboard profissional
- Gráfico de evolução
- Avaliação física premium
- Histórico de avaliações
- Timer de rounds
- Feed de atividades
- Eventos esportivos
- Backup dos dados
- PWA preparado
