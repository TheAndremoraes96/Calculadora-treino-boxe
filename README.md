# Calculadora-treino-boxe
Calculadora para atletas e praticantes de boxe com IMC, cronometro de rounds e ferramenta de treino.
## Funcionalidades

- [x] Cálculo de IMC
- [x] Classificação de IMC
- [x] Cronômetro de rounds
- [x] Contador automático de rounds
- [ ] Descanso automático
- [ ] Campainha de início e fim
- [ ] Contador de golpes

## Sobre o Projeto

Este projeto foi desenvolvido para auxiliar atletas e treinadores de boxe.
